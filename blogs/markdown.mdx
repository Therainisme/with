## 一、标题

支持六级标题

## 二、字体

**示例：**
```text
**这是加粗的文字**
*这是斜体的文字*
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```

**效果如下：**  

**这是加粗的文字**  
*这是斜体的文字*  
***这是斜体加粗的文字***  
~~这是加删除线的文字~~  

## 三、引用

**示例：**

```text
> 这是引用的内容
```

**效果如下：**  

> 这是引用的内容，里面应该能**加粗**
> 但是不能有标题

## 四、分割线

**示例：**

```text

```

## 六、超链接

**示例：**

[百度](http://baidu.com)  

## 七、表格

**示例：**  
```text
| 姓名 | 技能  | 排行 |

| 刘备 |  哭   | 大哥 |
| 关羽 |  打   | 二哥 |
| 张飞 |  骂   | 三弟 |
```

**效果如下：**  

| 姓名 | 技能  | 排行 |
| :-: | :-: | :-: |
| 刘备 |  哭   | 大哥 |
| 关羽 |  打   | 二哥 |
| 张飞 |  骂   | 三弟 |
